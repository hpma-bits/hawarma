# tests/ 目录架构

## 目录职责
测试模块，包含单元测试和集成测试，验证环境模拟器的正确性。

## 文件清单

| 文件名 | 状态 | 功能 |
|--------|------|------|
| `__init__.py` | ✅ | 包初始化 |
| `test_env_simulator.py` | ✅ | 环境模拟器测试套件 |
| `testset/` | ✅ | 测试图片集 |
| `output/` | ✅ | 测试输出 |

## 测试覆盖

1. **基础结构测试** - 事件创建、配方创建、状态初始化
2. **游戏时间限制测试** - 90 秒游戏时长、结束条件
3. **自动订单生成测试** - 4 秒间隔、槽位填充、刷新规则
4. **并行烹饪测试** - 多灶台同时工作
5. **积分系统测试** - 得分计算
6. **错误处理测试** - 无效输入、资源冲突
