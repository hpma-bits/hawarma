## 技术特点

### 使用的技术栈
- **Textual**: 现代 Python TUI 框架
- **asyncio**: 异步编程支持
- **Pydantic**: 数据验证和配置管理
- **Loguru**: 日志记录

### 架构特点
- **模块化设计**: 每个屏幕都是独立的组件
- **状态管理**: 通过应用类管理全局状态
- **异步支持**: 游戏运行支持异步操作
- **实时日志**: 游戏运行日志实时显示

## 使用方法

### 运行 TUI
```bash
python -m hawarma.tui
```

### 快捷键
- `q`: 退出应用
- `r`: 进入配方选择界面
- `c`: 进入配置面板
- `g`: 进入游戏控制界面

### 基本流程
1. 启动 TUI
2. 选择配方（点击"配方选择"按钮）
3. 配置设置（可选，点击"配置设置"按钮）
4. 开始游戏（点击"开始游戏"按钮）
5. 查看游戏日志（在游戏控制界面查看）
