# Hawarma TUI 使用说明

## 简介

Hawarma TUI 是一个基于 Textual 框架的文本用户界面，为烹饪游戏自动化 Agent 提供完整的仪表板界面。

## 功能特性

- **菜单栏**: 顶部菜单，包含所有功能入口
- **配方选择界面**: 可视化的配方选择，支持多选
- **配置面板**: 可编辑所有配置，包括基本设置、屏幕设置、匹配设置、游戏设置和调试设置
- **游戏控制界面**: 开始、暂停、停止游戏的控制按钮
- **日志区域**: 实时显示游戏运行日志

## 安装依赖

确保已安装 Textual 依赖：

```bash
uv pip install textual>=0.40.0
```

或者使用 requirements.txt：

```bash
uv pip install -r requirements.txt
```

## 运行 TUI

```bash
python tui.py
```

## 快捷键

- `q`: 退出应用
- `r`: 进入配方选择界面
- `c`: 进入配置面板
- `g`: 进入游戏控制界面

## 界面说明

### 主菜单

主菜单提供四个主要功能按钮：
- **配方选择**: 选择要使用的配方
- **配置设置**: 编辑应用配置
- **开始游戏**: 进入游戏控制界面
- **退出**: 退出应用

### 配方选择界面

- 显示所有可用配方
- 点击配方进行选择
- 点击"确认选择"保存选择
- 点击"返回"回到主菜单

### 配置面板

配置面板包含五个标签页：

1. **基本设置**
   - 图片目录
   - 日志目录
   - 配方数据路径
   - 游戏时长
   - 灶台类型

2. **屏幕设置**
   - 分辨率
   - 是否保存截图

3. **匹配设置**
   - 匹配策略
   - 匹配阈值

4. **游戏设置**
   - 灶台保留时间
   - Rush红色阈值
   - 策略选择

5. **调试设置**
   - 是否保存订单截图
   - 是否保存组装验证截图

### 游戏控制界面

- **开始游戏**: 开始运行游戏
- **暂停游戏**: 暂停游戏运行
- **停止游戏**: 停止游戏运行
- **日志区域**: 显示游戏运行日志

## 注意事项

1. 在开始游戏前，请先选择配方
2. 配置修改后需要点击"保存配置"按钮
3. 游戏运行时无法修改配置
4. 所有日志会实时显示在游戏控制界面的日志区域

## 故障排除

### TUI 无法启动

1. 检查是否已安装 Textual 依赖
2. 检查 Python 版本是否 >= 3.10
3. 检查是否有语法错误

### 配方加载失败

1. 检查 `data/recipes.json` 文件是否存在
2. 检查文件格式是否正确

### 配置加载失败

1. 检查 `configs/config.yaml` 文件是否存在
2. 检查文件格式是否正确

## 开发说明

TUI 代码位于 `tui.py` 文件中，使用 Textual 框架构建。主要包含以下组件：

- `MainMenuScreen`: 主菜单屏幕
- `RecipeSelectionScreen`: 配方选择屏幕
- `ConfigScreen`: 配置屏幕
- `GameControlScreen`: 游戏控制屏幕
- `HawarmaApp`: TUI 应用主类

如需修改或扩展 TUI 功能，请参考 Textual 官方文档：https://textual.textualize.io/
